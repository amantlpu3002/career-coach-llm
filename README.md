# CareerMind — AI Career Coach

[![Live Demo](https://img.shields.io/badge/Live%20Demo-HuggingFace-blue)](https://AmanT420-career-coach-llm.hf.space)

Fine-tuned **TinyLlama-1.1B** deployed as a Flask web app acting as an AI career coach.

## Features
- 📊 Resume Score — ATS scoring with improvement tips
- 🔑 Keyword Gap — finds missing keywords vs job description
- ✍️ AI Rewrite — rewrites your professional summary
- 🎯 Interview Prep — generates tailored interview questions

## Stack
QLoRA · TinyLlama-1.1B · Flask · HuggingFace Spaces · Docker · Python 3.11
